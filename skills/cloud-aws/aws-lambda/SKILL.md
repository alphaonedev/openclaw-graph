# Aws Lambda

## Purpose
AWS Lambda is a serverless computing service that executes code in response to events without managing servers.

## When to Use
- Running event-driven applications like API backends
- Processing data from AWS services such as S3 uploads
- Automating tasks without provisioning infrastructure

## Key Capabilities
- Execute code in multiple languages on demand
- Automatically scale based on traffic
- Integrate with other AWS services for workflows

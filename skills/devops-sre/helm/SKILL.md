# Helm

## Purpose
Helm is a package manager for Kubernetes that allows defining, installing, and upgrading applications via charts.

## When to Use
- Managing Kubernetes applications with charts
- Deploying and upgrading software on Kubernetes clusters
- Packaging applications for easy sharing and reuse

## Key Capabilities
- Package and deploy Kubernetes applications using charts
- Manage application versions and perform rollbacks
- Handle dependencies and configurations for Kubernetes resources

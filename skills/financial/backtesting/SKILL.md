# Backtesting

## Purpose
Test trading strategies on historical data to evaluate performance, risks, and profitability.

## When to Use
- When testing trading strategies on past market data
- To evaluate strategy performance before live trading
- For risk assessment in quantitative finance

## Key Capabilities
- Simulate strategies using historical financial data
- Calculate performance metrics like returns and Sharpe ratio
- Detect potential flaws or overfitting in strategies

# Memory Forensics

## Purpose
Analyzes volatile memory dumps to detect malware, rootkits, and security breaches in digital forensics.

## When to Use
- Investigating malware infections in RAM
- Detecting hidden processes or rootkits
- Analyzing live system artifacts during incidents

## Key Capabilities
- Extract and parse memory dumps
- Identify malicious code in memory
- Recover deleted or encrypted data from RAM

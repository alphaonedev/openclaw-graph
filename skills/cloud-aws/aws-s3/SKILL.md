# Aws S3

## Purpose
Interact with AWS S3 for object storage, managing buckets, objects, and access controls.

## When to Use
- Storing and retrieving unstructured data like files and images
- Hosting static websites with high scalability
- Archiving backups for cost-effective data retention

## Key Capabilities
- Upload, download, and delete objects in S3 buckets
- Create, configure, and manage S3 buckets and policies
- Generate presigned URLs for secure object sharing

# Terraform

## Purpose
Tool for defining, provisioning, and managing infrastructure as code using declarative HCL configuration files.

## When to Use
- Provision and manage cloud infrastructure declaratively
- Automate deployment of resources across multiple providers
- Version control and collaborate on infrastructure changes

## Key Capabilities
- Define infrastructure using HCL files
- Provision resources on providers like AWS, Azure, and GCP
- Manage state and detect configuration drifts

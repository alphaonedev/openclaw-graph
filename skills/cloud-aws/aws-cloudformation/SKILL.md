# Aws Cloudformation

## Purpose
AWS CloudFormation is a service for modeling and provisioning AWS resources using YAML or JSON templates.

## When to Use
- Automate deployment of AWS resources
- Manage infrastructure as code for consistency
- Version control and track changes to cloud setups

## Key Capabilities
- Define and provision AWS resources via templates
- Manage resource stacks for grouping and updates
- Support rollbacks and automated deployments

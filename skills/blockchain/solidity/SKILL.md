# Solidity

## Purpose
Solidity is a statically-typed language for writing secure smart contracts on the Ethereum blockchain.

## When to Use
- Writing Ethereum smart contracts
- Developing decentralized applications (dApps)
- Creating and managing tokens

## Key Capabilities
- Define contracts with functions and variables
- Handle events and state changes
- Implement inheritance and modifiers

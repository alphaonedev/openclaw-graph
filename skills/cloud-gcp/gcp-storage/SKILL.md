# Gcp Storage

## Purpose
Interact with Google Cloud Storage to manage buckets, objects, and access controls for scalable data storage.

## When to Use
- Storing and retrieving unstructured data like images or backups
- Serving static content for web applications
- Archiving data with cost-effective storage classes

## Key Capabilities
- Upload download and delete objects in buckets
- Create manage and delete storage buckets
- Set access policies and lifecycle rules

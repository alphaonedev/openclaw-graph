# Prometheus

## Purpose
Prometheus is an open-source monitoring and alerting toolkit for collecting metrics from targets and generating alerts.

## When to Use
- Monitoring application metrics in Kubernetes or cloud environments
- Setting up alerts for system or service failures
- Querying time-series data for performance analysis

## Key Capabilities
- Collect metrics via HTTP from various targets
- Store and query time-series data efficiently
- Define and trigger alerts based on custom rules

# Godot

## Purpose
Godot is an open-source game engine for 2D and 3D development using GDScript.

## When to Use
- Developing 2D or 3D games
- Prototyping game concepts quickly
- Building cross-platform applications

## Key Capabilities
- Node-based scene system for game structure
- GDScript and C# scripting support
- Integrated physics and animation tools
